# VRPhoto
使用陀螺仪传感器、四元数 通过计算偏移的位置，切换相应的图片，产生视觉移动，这就是一个简单的VR版本

展示一组图片，图片会根据陀螺仪方面改变显示相应的图片，让你感觉图片跟着我们的视觉在改变，一个VR图片的简版！
详细介绍请看博客：http://www.cnblogs.com/xiaoxiaoqingyi/p/6932008.html
